# AIM2019
The testing code and pretrained model for AIM Workshop and Challenge @ ICCV 2019.

## Runtime environment

* Python : 3.6.8
* Pytorch : 1.1.0
* CUDA : 9.0


## Data

* Place all the test images to "./DIV2K/DIV2K_test_LR_bicubic/".
* Or modify the "testsets" and "testset_L" in test_demo.py.
